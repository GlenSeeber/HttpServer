This is an HTTP server. The protocol for this server lies within the bounds of the HTTP Protocol.

The server accepts GET and POST currently.

For all requests:

	- Header of no more than 4096 bytes.
	- Must end in "\r\n\r\n" (as is already required by http)

GET:
	
	- Only the header of the request will be read.
	- Server response will lack a header

POST:

	- Must include "Content-Length: [n]" (case sensitive)
	- Server will respond with 200 OK if it has recieved the payload.
